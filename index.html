2. Introduction

This project explores which in-game team statistics best predict the outcome of an NBA game.
By analyzing historical NBA data, we aim to answer: What performance factors most strongly influence team wins?

Understanding these factors can assist analysts, coaches, and fans in interpreting what truly drives success beyond raw points scored.
We will build predictive models and use explainable ML techniques to back our findings.

3. Data Curation

We obtained data from the NBA Dataset on Kaggle which includes game-level and team-level stats from 19462022.
We preprocessed it into a tidy format where each row represents one team in one game, and includes whether the team won.

The original datasets include game.csv, team.csv, and others. Cleaning included:

  • Importing CSVs
  • Parsing numeric and datetime columns
  • Reshaping to team-game level
  • Adding engineered features like win_flag, FG%, REB, and TOV

We chose the NBA Database from Kaggle because it offers a detailed collection of team and player statistics across thousands of games, making it ideal for hypothesis testing, visualizations, and predictive modeling.

def clean_dataframe(df, df_name, critical_columns=[]):
    """
    Cleans a DataFrame by:
      - Dropping columns with >90% missing values.
      - Imputing numeric columns with their median.
      - Imputing categorical columns with their mode.
      - Converting columns containing 'date' in their name to datetime.
      - Dropping rows missing values in critical columns.
    """
    drop_threshold = 0.9
    cols_to_drop = [col for col in df.columns if df[col].isna().mean() > drop_threshold]
    df.drop(columns=cols_to_drop, inplace=True)

    for col in df.columns:
        if pd.api.types.is_numeric_dtype(df[col]):
            if df[col].isna().sum() > 0:
                df[col] = df[col].fillna(df[col].median())
        else:
            if df[col].isna().sum() > 0:
                mode_val = df[col].mode()[0] if not df[col].mode().empty else 'Unknown'
                df[col] = df[col].fillna(mode_val)
        if 'date' in col.lower():
            df[col] = pd.to_datetime(df[col], errors='coerce')

    if critical_columns:
        df.dropna(subset=critical_columns, inplace=True)

    return df

5. Primary Analysis: Random Forest Classification

In this section, we build and tune a Random Forest classifier to predict whether the home team wins based on its in-game performance statistics. We begin by selecting numeric features (e.g., field goals, free throws, rebounds, turnovers) and removing any columns that would leak the game outcome.

We then split the data by seasontraining on seasons before 2020 and testing on 2020 and beyondto avoid temporal leakage. Hyperparameters (n_estimators, max_depth, min_samples_leaf, max_features) are optimized via 10-fold cross-validation using ROC AUC as our scoring metric. Finally, we evaluate the best model on the held-out test set and analyze its performance (accuracy and AUC) as well as feature importance.

Discussion of Results

The tuned Random Forest (1000 trees, max_depth=10, 50% feature subsampling, leaf size=5) achieved a test accuracy of 78.1% and a test ROC AUC of 0.854, closely matching its cross-validated AUC of 0.816. This indicates good generalization to unseen seasons without overfitting.

We trained a Random Forest classifier using 10-fold cross-validation to predict home-team wins based on performance metrics. The best model used 1,000 trees, a max depth of 10, and a 50% feature subsample, achieving a cross-validated AUC of 0.816. On the held-out test seasons (2020+), the model reached 78.1% accuracy and a ROC AUC of 0.854, indicating strong generalization. Feature importance highlighted field goal percentage, rebounds, and free throws made as the top predictorsunderscoring that scoring efficiency and rebounding drive home-court success.

7. Insights and Conclusions

Our analysis shows that home-team performance in the NBA is driven primarily by shooting efficiency and rebounding. In both the exploratory phase and our Random Forest model, field goal percentage and rebounds (especially defensive) emerged as the strongest predictors of winning.

Interestingly, while total points scored (e.g., field goals made) clearly contribute to a win, it was not the top drivershooting efficiency (fg_pct_home) outweighed sheer scoring volume. This suggests teams gain a competitive edge not just by outscoring opponents, but by maximizing accuracy.

The Random Forests test AUC of 0.854 and balanced confusion matrix further demonstrate the reliability of these metrics for predicting game outcomes.

Limitations:

  • We only used box-score statistics; incorporating lineup data or player-level features could improve accuracy.
  • Seasonal rule changes (e.g., pace of play) may affect transferability to future seasons.

Future Work:

  • Extend the model to multi-season rolling forecasts.
  • Explore clustering to identify styles of home-court advantage.
  • Integrate betting-market odds to assess predictive power in a live setting.
